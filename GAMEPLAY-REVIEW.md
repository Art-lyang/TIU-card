## 2. Act 구조 현황

### 4-Act 전환
| 항목 | 상태 | 비고 |
|------|------|------|
| Act 1 프롤로그/일상 (~5일) | ✅ | 4장/일, 이변체 없음, 현장임무 없음 |
| Act 1→2 전환 (~5일) | ✅ | 무조건 전환, 루트 A 고정 |
| Act 2 의혹 → Act 3 진실 전환 (~14일) | ✅ | 무조건 전환 |
| Act 3 진실 → Act 4 최종 전환 (~29일) | ✅ | 무조건 전환 |
| Act 1 간부진 자기소개 (ORACLE 의심 없음) | ✅ | 인사+업무보고 톤 |
| Act 2부터 ORACLE 의심 시작 | ✅ | 서하은·윤세진·임재혁 감지 |
| 4루트 분기 (A/B/C/D) | ✅ | actFlags (prom_met, mission_done 등) |
| 전환 루트 전용 카드 (CT-001~011) | ✅ | transReq 필드 |
| ORACLE 브리핑 화면 | ✅ | 루트별 텍스트 분기 |
| Act 3 진입 시 전체 -3 | ✅ | 전환 시 stat 적용 |
| Act 4 진입 시 전체 -5 | ✅ | 전환 시 stat 적용 |
