# TIU-card 게임플레이 리뷰
## Full Gameplay Review & Game-Feel Assessment

**리뷰 시점**: 시설 확장 시스템 + 커버 스토리 LOG 구현 완료 후
**평가 기준**: 게임성, 몰입감, 리플레이 가치, 밸런스, UX

---

## 1. 전체 게임 루프 평가

### 핵심 루프
`부팅 → 튜토리얼 → [카드 스와이프 × cpd] → 뉴스 → 리워드 → 저녁 대화 → 다음 날`

**강점:**
- 루프가 명확하고 한 사이클이 짧아서 "한 판 더" 중독성이 있음
- Act별 카드 수 증가(5→6→7)가 자연스러운 난이도 상승을 만듦
- 뉴스 → 리워드 → 저녁 대화 순서가 정보 수집 → 결정 → 여운의 감정 흐름을 잘 만듦

**개선 가능 포인트:**
- 매일 동일 구조라 10일 이후 다소 예측 가능해짐
- 시설 확장 카드가 이 반복감을 깨는 좋은 변수가 될 수 있음 (현재 30% 확률로 적절)

**게임성 점수: 7.5/10**

---

## 2. 카드 밸런스

### 스탯 변동
- `applyFx` 배율 ×5: fx 값 1 = 실제 5 변동. 이 스케일이 적절함
- 초기 스탯 c:50, r:65, t:50, o:40 — 자원이 높고 ORACLE 평가가 낮은 것이 "독립적 운영 vs 순종" 긴장감을 잘 만듦

### GI (Global Index) 시스템
- 양수 = ORACLE 순응, 음수 = 독립적
- gi >= 60이면 A엔딩(ORACLE에 의한 "해방"): 엔딩 다양성에 기여
- 대화 선택에서 gi 변동이 크다는 게 좋은 디자인 — 대화의 무게감이 느껴짐

### 리스크 시스템 (RISK_MSG)
- 보급 카드에서만 20% 확률 실패: 서프라이즈 요소가 적절
- "물자 상태 불량" 등 메시지가 세계관에 맞음

### Act 전환 밸런스
- Act 1→2: Day 10에 전환, 루트 A~D로 분기 — 적절
- Act 2→3: Day 25에 전환 — Act 2가 15일로 가장 길어서 중반부 피로감 가능성
- Act 2~3 감쇠(c/r -1~2): 후반부 생존 자체가 도전이 됨, 좋은 긴장감

**밸런스 점수: 8/10**

## 3. 서사 & 몰입감

### 강점
- ORACLE의 이중성이 점진적으로 드러남 (데이터 누락 → 삭제 → 감시 → 조작)
- 캐릭터 신뢰도 시스템이 서사 깊이를 만듦 (trust 65/70 이상 시 핵심 대화 해금)
- 서하은의 전출과 박소영 합류가 감정적 전환점 역할
- 프로메테우스가 적→?의 구도로 반전이 효과적
- LOG 시스템이 "발견"의 쾌감을 줌

### 시설 확장이 서사에 미치는 영향
- 확장 카드의 스토리 텍스트가 캐릭터 기반 ("윤세진이 보고", "강도윤 건의")이라 서사와 자연스럽게 연결
- 커버 스토리 LOG가 세계관 디테일을 강화: "설악 콜드체인" 같은 민간 위장이 현실감을 높임
- 시설 변화가 눈에 보이는 결과물이라 선택의 무게가 올라감

### 개선 가능 포인트
- 대화에서 fx를 제거한 것(g/trust만)이 대화의 순수한 관계성을 강조하는 좋은 결정
- 다만 일부 플레이어는 "대화 선택이 게임에 영향을 안 주나?"라고 느낄 수 있음
  → trust 임계값 대화 해금과 특수 엔딩 연동이 이를 보상하지만, 피드백이 즉각적이지 않음

**서사 점수: 9/10**

## 4. 시설 확장 시스템 평가

### 현재 구현된 흐름
1. 게임 중 30% 확률로 확장 제안 카드 등장 (하루 1회 제한)
2. 우스와이프 = 승인 → 리워드 풀에 추가, 좌스와이프 = 대기 → 나중에 승인 가능
3. 리워드 페이즈에서 확장 리워드 선택 시 → 완료, 새 카드 영구 추가
4. 뉴스 페이즈에 시설 상태 섹션 표시

### 강점
- "승인했다고 바로 적용되는 게 아니라 리워드에서 선택해야 한다"는 2단계 구조가 전략적 깊이를 만듦
- 확장 완료 후 관련 카드가 풀에 추가되어 세계가 확장되는 느낌
- 대기 목록(pending)에서 언제든 승인 가능하여 선택의 유연성 보장
- 리워드 fx가 기존 리워드와 유사한 수준이라 밸런스 유지

### 주의 사항
- 확장 8개 + 새 카드 11장: 컨텐츠 양이 적절. 더 늘리면 관리 복잡도 상승
- Act 2에서만 등장하는 확장(FE-005~007)이 후반부 변화를 만드는 좋은 구조
- 시설도 페이지와의 시각적 연동(Phase 4)이 아직 미구현 → 향후 가장 임팩트 큰 개선

**시설 시스템 점수: 8/10** (시설도 시각 연동 완료 시 9/10 예상)

## 5. UX & 인터페이스

### 강점
- 스와이프 기반 조작이 직관적
- 카드에 fx 힌트(↑↓ 화살표)가 있어 의사결정 정보 제공
- Act별 UI 컬러 변화가 분위기 전환 역할
- 모바일 터치 대응 완비
- 토스트 메시지(리스크, 시설 승인)가 즉각 피드백 제공

### 개선 가능 포인트
- info-bar 태그가 많아지고 있음 (ACT, 카드, LOG, 아카이브, 시설) → 모바일에서 줄바꿈 발생 가능
- 시설 확장 카드의 hint 텍스트가 파란색으로 기존 카드와 시각적 차별화가 되지만, "이것은 일반 카드가 아니다"라는 인식이 처음에 부족할 수 있음
  → 카드 헤더에 "FACILITY PROPOSAL" 같은 태그 추가 고려

**UX 점수: 7.5/10**

## 6. 리플레이 가치

### 현재 리플레이 요소
- 엔딩 수집 (다수 엔딩 + 특수 엔딩)
- LOG 전체 해금
- 아카이브 수집
- 신뢰도 루트 분기 (각 캐릭터별)
- Act 전환 루트 4가지 (A~D)
- 세션 카운트

### 시설 확장이 추가한 리플레이 요소
- 8개 확장 중 어떤 것을 우선 완료하느냐에 따라 카드 풀이 달라짐
- 확장 조합에 따른 플레이 스타일 분화 가능성

**리플레이 점수: 8/10**

## 7. 종합 평가

| 항목 | 점수 |
|------|------|
| 게임성 | 7.5 |
| 밸런스 | 8.0 |
| 서사 | 9.0 |
| 시설 시스템 | 8.0 |
| UX | 7.5 |
| 리플레이 | 8.0 |
| **종합** | **8.0/10** |

### 핵심 강점 3가지
1. **서사 품질**: ORACLE의 이중성, 캐릭터 관계, 세계관 디테일이 SCP류 게임 중에서도 높은 수준
2. **선택의 무게**: 카드마다 진짜 고민이 되는 트레이드오프가 있음
3. **시설 확장의 가시성**: 내 선택이 기지를 물리적으로 변화시킨다는 감각이 강력함

### 가장 효과적인 개선 3가지
1. **시설도 시각 연동** (Phase 4): 확장 완료 시 시설도에 방이 추가/변화되는 것이 보이면 임팩트 극대화
2. **mid-game 변주**: Day 10~25 구간에 특별 이벤트(이변체 침공, 프로메테우스 접선 등)가 더 있으면 중반 피로감 해소
3. **즉각 피드백 강화**: 대화에서 trust 변화 시 짧은 시각 피드백 ("서하은과의 신뢰 ▲" 등)
