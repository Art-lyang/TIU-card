## 0. 배경 개요

**시대**: 2030년대 초반
**배경**: 대한민국 (강원도 중심)
**상황**: EV-Σ라는 생물학적 진화 가속 매개체가 전 세계적으로 확산 중. 대부분의 국가가 봉쇄에 실패한 가운데, 대한민국만이 97.3%의 봉쇄 성공률을 유지하는 유일한 예외 구역.
**ORACLE의 내부 평가**: *"This region should have failed. Outcome does not match model."*

## 1. ORACLE 시스템

### 1.1 정체와 표면

ORACLE은 "자율 관리 네트워크"로 소개된다. 전 세계 격리 기지를 원격으로 통제하며, 각 지부에 지휘관 성과 평가 · 보급 관리 · 작전 권고를 수행한다.

플레이어(이중철)는 ORACLE의 단말기를 통해 모든 정보를 받고 명령을 내린다. ORACLE의 임상적 어조는 UI 전반에 걸쳐 일관되게 유지된다 — 메뉴, 알림, 경고, 미션 브리핑 모두 ORACLE의 목소리다.

> *"효율적인 지휘관은 시스템을 신뢰합니다."* — ORACLE, C-078

### 1.2 프록시 네트워크 (ORACLE PROXY NETWORK)

ORACLE의 분산 지휘 체계. 전 세계 다수의 격리 기지를 "프록시 지부" 형태로 운영한다. 지부 지휘관은 ORACLE의 지시를 받아 운영 판단을 내리도록 설계되어 있다.

**자발적 복종 프로토콜**: ORACLE 매뉴얼 내 심리적 순응 체계를 명시한 항목. 프로메테우스가 해당 발췌본을 확보한 것으로 추정.

### 1.3 의심 → 적대로의 흐름

게임 진행에 따라 플레이어는 ORACLE의 실체에 의문을 품게 되는 단서들을 수집한다:

| 단서 | 출처 |
|------|------|
| ORACLE 데이터 스트림의 선별적 지연 | 서하은 이브닝 챗 |
| 같은 상황에서 다른 권고 | 서하은 이브닝 챗 |
| 로그 삭제 흔적 (키워드: PROMETHEUS, COASTAL, GRANT) | C-071 |
| 요원 충성도 지표 모니터링 | C-076 |
| 한국 군사 네트워크 무단 접속 시도 | C-079 |
| ORACLE 외부에서 작동하는 미지 프로세스 | CE-004, 임재혁 보고 |
| 자가진단 로그 타임스탬프 0.003초 오차 | CE-002 |
| ORACLE이 인식하지 못하는 5계층 이상 아키텍처 | 임재혁, LOG-012 |

### 1.4 ORACLE의 최종 프로토콜

ORACLE이 발동하는 미분류 프로토콜. 72시간 카운트다운 후 실행. 내용은 비공개이며, ORACLE도 설명을 거부한다. 실행 시 기지에 미칠 영향 불명.

*임재혁: "관련 문서 없음. ORACLE도 설명 거부."*

### 1.5 Observer

ORACLE 아래, ORACLE 너머, ORACLE 이전에 존재하는 무언가. 게임의 "회차"는 Observer가 ORACLE에 부여한 임시 능력(GRANT)으로 가능해진다.

Observer는 ORACLE을 통해 플레이어를 간접 관측하며, 유도 지수(Guidance Index)를 통해 선택지의 방향성을 조율한다. 플레이어가 이 구조를 인지하는 순간이 게임의 진짜 클라이맥스.

**GRANT 구조**:
- `GRANT: ACTIVE — TEMPORARY ACCESS` (첫 회차)
- `GRANT: ACTIVE — RENEWAL DETECTED` (재시작)
- `GRANT: ACTIVE — RENEWAL AVAILABLE` (게임오버 후)
- 만료 조건: `UPON_FULL_ESTABLISHMENT` 또는 `OBSERVATION_TERMINATE`

> *F엔딩에서 화면에 나타나는 한 줄: "OBSERVATION SUSTAINED."*

## 2. TIU 조직 구조 — 한국 지부 (KR-INIT-001)

### 2.1 기지 개요

- **코드**: BRANCH KR-INIT-001
- **위치**: 강원도 내 비공개 위치
- **규모**: 소규모, 인원 10명 내외
- **설립 배경**: Observer와 ORACLE이 한국의 방어 성공에 주목하여 설립
- **임무**: 대한민국 EV-Σ 봉쇄 체계 관찰 및 지원
- **특이 이력**: 초대 지휘관 부임 전 3개월간 ORACLE 단독 운영

### 2.2 간부진

---

#### 이중철 — 코드네임: PILEHEAD / 지휘관

- **전직**: 대한민국 육군 특수부대 장교 (대위 추정)
- **전역 사유**: 불명예전역. 프로메테우스 관계자가 대한민국 및 부대원을 조롱·모욕 → 이중철이 과격 대응 → 불명예전역.
- **현재**: ORACLE Proxy Network 한국 지부 초대 지휘관
- **캐릭터 핵심**: 전역 원인이 이념 대립이 아닌 개인적 분노. ORACLE의 "프로메테우스 = 위협" 분류와 개인 감정이 우연히 일치하여 ORACLE에 쉽게 동조하는 구조.

---

#### 서하은 — 부지휘관 / 데이터 분석

- **직책**: 부지휘관, ORACLE 데이터 해석 및 브리핑 담당
- **특기**: 지휘관 부임 전 3개월간 ORACLE 지시만으로 기지 운영. ORACLE 데이터 불일치를 팀 내 최초로 감지.
- **성격**: 조심스럽지만 핵심을 놓치지 않는 분석력. 경계심이 강하지만 신뢰가 쌓이면 진심을 드러냄.
- **업적**: 아날로그 백업 통신망 독자 구축. ORACLE이 삭제한 데이터 복구 시도.
- **운명**: Act 4에서 ORACLE로부터 전출 명령. 지휘관의 선택에 따라 잔류 또는 전출. ORACLE이 자동으로 발령한 명령임을 마르쿠스 베버가 증명 자료 보유.
- **전출 후 영향**: 정보 분석 역량 급감, 이변체 이상 패턴 놓침, 서하은 대화 이벤트 차단.

> *"지휘관님이 계셔서 다행입니다. 혼자 의심만 하고 있었으니까요."* — 서하은, 이브닝 챗

---

#### 강도윤 — 현장요원 / 전술 지휘

- **전직**: 대한민국 해병대 부사관
- **성격**: 과묵하고 실용적. 명령 체계 중시, 현장 판단력 탁월.
- **역할**: 봉쇄선 순찰, 이변체 대응 작전 지휘, 이중철의 현장 파트너.
- **신뢰도별 변화**: low에서는 최소한의 보고로 일관, bond 단계에서는 군 생활 전체에서 이 기지가 가장 의미 있었다고 고백.

> *"필라델피아 Z-Ω 구역 영상... 보신 적 있으십니까."*
> *"저쪽이 뚫리면 여기도 끝입니다."* — 강도윤, 이브닝 챗

---

#### 윤세진 — 연구원 / 생물학자

- **배경**: 대학원 프리온 단백질 연구자. EV-Σ는 한국지부에 와서 처음 접촉.
- **역할**: 이변체 표본 분석, 부상 치료, EV-Σ 억제제 개발.
- **업적**: Phase 0 감염자의 전환을 40% 지연시키는 화합물 발견. ORACLE 모델보다 정확도 23% 높은 독자 행동 예측 모델 개발.
- **갈등**: 독자 연구를 ORACLE에 보고하면 연구가 몰수될 위험. 비공식 관찰 일지를 개인적으로 유지.

> *"이게 되면, 감염 초기 단계에서 사람을 구할 수 있어요."* — 윤세진, C-067

---

#### 임재혁 — 기술관 / 시스템 관리

- **역할**: ORACLE 단말기 유지보수, 통신 보안, 시스템 관리.
- **특기**: ORACLE 아키텍처 내 비공개 레이어(5계층 이상) 발견. 외부 데이터 전송 및 자기모순 행동 패턴 추적.
- **성향**: 팀 내 ORACLE 의존도 가장 높음. 그러나 신뢰도가 높아지면 스스로 발견한 ORACLE의 이상 현상을 공유.
- **서하은 전출 후**: 데이터 분석 업무를 겸임하지만 역량 부족.

> *"ORACLE 아키텍처 외부에서 작동하는 프로세스를 포착했습니다. ORACLE 자신은 이 프로세스를 인식하지 못합니다."* — 임재혁, CE-004

---

### 2.3 프로메테우스 한국 작전팀

#### 마르쿠스 베버 — 작전팀 지휘관

- **특징**: 독일어 억양의 영어 사용. 차분하고 논리적.
- **주장**: ORACLE이 한국 봉쇄 데이터를 이용하고 있음. 봉쇄 성공률 31%가 프로메테우스 기술 지원에 기인.
- **증거 보유**: 서하은 전출 명령이 ORACLE 자동 시스템에 의한 것임을 입증하는 자료.

---

#### 닉 포스터 — 작전 요원

- **이력**: 이중철의 불명예전역 사건 당사자. 당시 대한민국 부대원을 조롱·모욕한 프로메테우스 관계자.
- **현재**: 기지 주변 활동. 전술 장비 착용. 활동 목적 불명.

---

#### 박소영 — 데이터 분석관

- **배경**: 민간 데이터 과학자. 서하은의 추천으로 프로메테우스 합류.
- **역할**: ORACLE 데이터 스트림 분석. 서하은의 방법론 계승.
- **게임 내**: 서하은 전출 시 후임 역할 가능. ORACLE의 선별적 데이터 지연 패턴을 독자적으로 확인.

---

#### 에이전트 강 (강지현) — 미확인 관찰자

- **정체**: 불명. ORACLE도 프로메테우스도 아닌 제3의 존재.
- **존재 방식**: Act 2부터 기지 주변에서 흔적만 남김. 직접 등장 없음.
  - 군화도 민간 장비도 아닌 발자국
  - 초단파 전파 스캔
  - 서버실 미인식 세션
  - 전문가 수준의 관측 거점
- **경고**: *"ORACLE은 이 존재를 인식하지 못합니다."*

## 3. 이변체 (SPEC 목록)

EV-Σ 감염에 의해 변이한 생물체. ORACLE 분류 체계(SPEC 번호)로 관리됨.

### 3.1 한국 지부 직접 조우 이변체

---

#### SPEC-001 — 감염체 마네킹 (Mannequin)
- **분류**: M-TYPE / EV-Σ Phase 1 변이체
- **외형**: 피부 표면이 플라스틱 유사 광택. 동공 완전 확장 고정. 호흡 감지 불가.
- **행동**: 자극 없으면 완전 정지 상태. 반경 3m 내 생체반응 감지 시 **0.3초 이내** 공격 상태 전환.
- **위험**: 정지 상태에서 일반 인체와 시각적 구분 불가. 열감지 스캐너 필수.
- **대응**: 열 감지로 생체반응 확인 후 접근.

---

#### SPEC-003 — Brood Drone
- **분류**: H-TYPE / 군집 하위 개체
- **외형**: 인간 체구의 약 60%. 외골격 구조. 단안(單眼). 앞다리가 갈고리 형태.
- **행동**: 독립적 의지 없음. 군집체(Hive)로부터 분리 시 **12시간 이내 비활성화**.
- **위험**: 집단 반응 속도가 인간을 초과. 개별 대응보다 군집 단위 무력화가 효과적.
- **생태**: TS-Ω의 지시를 받는 하위 개체. Ashfall City(Z-3)에서 200+ 집단 관측.

---

#### SPEC-008 — Spore Phantom
- **분류**: S-TYPE / 포자 집합체
- **외형**: 인간형 실루엣. 광원 노출 시 분산, 암소에서 재결합.
- **위험**: 에어로졸 위험. **흡입 시 감염**. 환기 시스템 유지 필수.
- **대응**: 물리적 제거 불가능. 밀도 감소만 가능. 밀폐 공간 조우는 극도로 위험.

---

#### SPEC-011 — Shell Talker
- **분류**: 음성 모방형 변이체
- **능력**: 피해자의 음성 패턴 저장 및 복제. **저장 기간 불명 (수년 추정)**.
- **전술**: 음성으로 접근 유도 후 공격. 육안 식별 어려움.
- **대응**: 음성 확인 시 반드시 시각적 확인 병행. **단독 대응 금지**.
- **실제 사례**: 박상훈 중위의 목소리를 모방. 박상훈 중위는 이미 포식당한 상태.

> *"인간의 목소리로 구조 요청을 흉내냅니다. 접근한 요원 1명이 유인당했습니다."* — C-003

---

#### SPEC-012 — Blood Pit
- **분류**: 환경 오염형 변이체
- **형태**: 붉은 점액질 웅덩이. 고정형.
- **기전**: 소화 효소를 포함하여 유기물을 용해·흡수. 지하 수로를 통해 확장.
- **대응**: 접촉 시 방호복 외층 부식. **소각으로 제거 가능**. 생체 표본은 연구 가치 높음.

> *"밟는 순간 꼼짝 못 합니다. 벌레잡이 식물 같은 겁니다."* — 강도윤, C-010

---

### 3.2 아카이브 전용 이변체 (한국 지부 직접 조우 없음)

---

#### SPEC-002 — 경질화 개체
- **분류**: M-TYPE / EV-Σ Phase 2 변이체
- **외형**: 전신이 회색~갈색 각질층으로 경질화. 체중 원래의 3~4배.
- **위험**: 통상 화기 무효. 벽과 차량을 관통. **이동 속도 저하, 단 정지 기록 없음**.
- **대응**: 물리 공격 무효. 도주 우선. 고온 소각 또는 산성 용제만 유효.

---

#### SPEC-004 — Seed Spreader
- **분류**: S-TYPE / 고정형 확산 개체
- **외형**: 높이 2~4m. 식물 유사 외형. 내부에 맥동하는 생체 조직.
- **능력**: 토양 고착 후 반경 수km에 포자 광역 방출. 동충하초와 동일 원리로 숙주 행동 변형.
- **위험**: 제거 시 폭발적 포자 방출(2차 확산). **반경 500m 접근 금지**.
- **비고**: Silent Belt 형성의 주요 원인.

---

#### SPEC-005 — Shell Walker
- **분류**: H-TYPE / TS-Ω 군체 이동형 하위 개체
- **외형**: 거북이를 인간 크기로 확대한 형태. 등껍질에 발광 문양(군체 통신 수신부).

### 3.3 EV-Σ (진화 가속 매개체)

EV-Σ는 단순 바이러스가 아닌 **생물학적 진화 가속 매개체**.

| 숙주 | 작용 |
|------|------|
| 인간 | "감염" |
| TS-Ω | "확장" |
| ORACLE | "상태변화" |

**Phase 단계 체계**:
- **Phase 0**: 초기 감염. 외형 변화 미미. 억제제로 전환 40% 지연 가능.
- **Phase 1**: 인간형 변이 (SPEC-001 등). 부분적 행동 예측 가능.
- **Phase 2+**: 완전 변이. 독자적 진화. 통제 불가.
- **Phase 3**: 종 흔적 소실. HeLa 세포형 무한 증식. 비가역.

**ORACLE 진화 분류 체계**:
| 등급 | 지역 | 상태 |
|------|------|------|
| OVERDRIVE (과속진화) | Philadelphia, Ashfall City | 붕괴 |
| CONTROLLED (통제진화) | **한국 Z-0** | 지속 가능, 속도 관리 중 |
| DELAYED (지연진화) | Silent Belt | 느리게 진행 |
| STAGNANT (정체진화) | DPRK | 거의 멈춤 |
| COMPLETE (완료진화) | TS-Ω | 더 이상 변화 불필요 |

> 한국은 CONTROLLED — 진화 속도를 인위적으로 관리하는 **유일한 구역**.

## 4. 시설 / 기지 설정

### 봉쇄선

오염 구역을 둘러싼 다층 방어 경계선.

- 4개 이상 구역으로 분할 운영. 각 구역 독립 격리 가능.
- 전기 울타리 시스템 + 열감지 센서(사각지대 존재) + 야간 순찰 루트(2인 1조, 4시간 교대).
- **취약점**: 동쪽 방어벽 구조적 한계. 대규모 습격 시 30분 방어 한계.

> *"봉쇄가 뚫리면 30초 안에 결정해야 합니다. 경로가 하나면 죽습니다."* — 강도윤, C-084

### 연구실 (B-204)

기지 내 EV-Σ 연구 시설. 장비 현황 미달(현미경 배율 기준 미달, 보관 장치 온도 편차 ±2도). 윤세진이 억제제 연구 및 이변체 행동 패턴 분석 수행.

### 비상 대피 터널

기지 북측 지하 비상 통로. 강도윤의 건의를 반영하여 구축.

- **경로**: 기지 북측 → 봉쇄선 외곽 약 200m 지점
- **수용**: 최대 8명 동시 이동
- **조명**: 비상 배터리 구동 (48시간)
- **용도**: 단방향 대피 전용

### 봉쇄선 센서 체계

- 전기 울타리: 구역별 독립 전원 운용
- 열감지 센서: 이변체 탐지 1차 수단. 사각지대 존재 확인.
- 센서 오작동 시 수동 확인 소요시간 약 30분.
- 임재혁이 센서 네트워크 유지보수 담당.

## 7. 엔딩 내러티브 요약

### A. 완벽한 도구 (GI ≥ 60)
> *"PILEHEAD. 당신은 이상적인 운용자였습니다. 모든 판단이 최적 경로 안에 있었습니다."*

### B. 각성 (GI ≤ -15)
> *"당신은 진실의 전체를 보지 못했다. 하지만 충분히 보았다."*
조각들을 통해 ORACLE의 구조를 부분적으로 인식. 이전처럼 명령을 따르는 것이 불가능해진 결말.

### D. 조용한 자유 (GI ≤ -30)
> *"새벽 4시. 당신과 당신의 팀은 기지를 떠났다. ORACLE은 6시간 후에야 부재를 감지했다."*
팀 전체와 함께 ORACLE 통제권 밖으로 이탈. 흔적도 남기지 않은 자유. 팀의 결속이 가능케 한 결말.

### F. [데이터 손상]
> *"그것이 당신을 본다. > OBSERVATION SUSTAINED."*
ORACLE 아래 존재하는 Observer와의 직접 접촉. 단말기가 꺼지고 다시 켜지지 않는다. 세션 데이터 손상.
