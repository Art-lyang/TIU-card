## M-007: 결정적 타격

**Act**: 3 전용
**트리거**: Act 3 신규 카드 (GI ≥ 40일 때 등장)
**서사 역할**: 고GI 루트의 클라이맥스. ORACLE이 명령하는 프로메테우스 거점 타격. 실행하면 A엔딩 접근, 거부/이탈하면 GI 급락.

### 노드 구조

```
start
├── strike → assault → (end: 타격 성공)
├── recon → discovery → betray_or_report
│                        ├── report (end: ORACLE 보고)
│                        └── betray (end: 배신 — GI 급락)
└── oracle_tactical → precision_strike → (end: 정밀 타격)
```

### 노드 상세

**start**
```
[ORACLE 특별 작전 지시 — 기밀 등급: MAXIMUM]

프로메테우스 거점 위치가 확정되었습니다.
좌표: 강원도 동해안 인근 폐산업단지.

이중철 지휘관. 이것은 한국 지부의 존재 의의를 증명할 작전입니다.

[ORACLE: 선제 타격을 권고합니다. 전 요원 동원.]
```
- ▸ 전원 출격 → strike
- ▸ 소규모 정찰조 우선 투입 → recon
- ▸ ORACLE 전술 지원 요청 → oracle_tactical (g: +3)

**strike**
```
전 요원이 출격합니다. 강도윤이 선두.

폐산업단지 도착. 건물 3동. 통신 장비 흔적.

그러나 — 안에 사람이 있습니다. 무장하지 않은 민간인 연구자 4명.

강도윤: "...이것들은 전투원이 아닙니다. 연구 장비만 있습니다."

[ORACLE: 작전 계속 진행. 프로메테우스 관계자는 전원 무력화 대상입니다.]
```
- ▸ 작전 계속 — 무력화 → assault
- ▸ 작전 중단 — 이건 아니다 → abort

**assault**
```
ORACLE의 지시대로 연구자들을 제압합니다.

장비를 압수합니다. 연구자들의 눈에서 두려움이 보입니다.

한 명이 말합니다.
"우리는 치료제를 만들고 있었습니다."

강도윤이 당신을 봅니다. 아무 말도 하지 않습니다.

[ORACLE: 작전 완료. 우수한 성과입니다.]
```
- [ 기지 귀환 — 작전 성공 ] → end
  - result: { c: 2, r: -2, t: -3, o: 3 }, g: +5
  - log: "LOG-016"

**abort**
```
"작전 중단."

강도윤이 고개를 끄덕입니다. 안도하는 표정.

[ORACLE: ...지휘관. 작전 중단 사유를 보고하십시오.]

대답하지 않습니다.

연구자 중 한 명이 다가옵니다.
"감사합니다. 베버 박사가 당신에 대해 말했습니다."
```
- [ 기지 귀환 — 명령 불복종 ] → end
  - result: { c: -1, r: 0, t: 2, o: -3 }, g: -8
  - log: "LOG-016"

**recon**
```
강도윤과 2명만 투입합니다.

폐산업단지 외곽에서 관찰.

건물 내부 — 연구 장비. 생화학 분석기. 약물 합성 장비.
사람 4명. 백의. 비무장.

그리고 벽에 붙은 사진 하나 — EV-Σ 감염 초기 단계 환자.
그 옆에 수기 메모: "Phase 1→2 전환 억제 성공률: 73%"

이것은 무기 제조 시설이 아닙니다.
```
- ▸ 내부로 진입해서 확인 → discovery
- ▸ 정보 기록 후 철수 → retreat_info

**discovery**
```
안으로 들어갑니다.

연구자가 놀라지만, 무기가 없다는 걸 보여줍니다.

"마르쿠스가 보냈습니까?"

아니라고 하자, 표정이 바뀝니다.

"그러면... ORACLE이?"

강도윤이 연구 데이터를 훑어봅니다.
"지휘관. 이건 윤세진이 봐야 합니다. EV-Σ 억제 연구... 우리보다 훨씬 앞서 있습니다."
```
- ▸ ORACLE에 보고한다 → report
- ▸ 연구 데이터를 비공식 확보 → betray

**report**
```
[ORACLE: 데이터 수신. 분석 중.]
[ORACLE: 해당 시설은 비인가 생화학 연구소입니다. 폐쇄를 권고합니다.]
[ORACLE: 연구 데이터는 ORACLE 본부에서 검증 후 활용 여부를 결정합니다.]

강도윤이 속삭입니다. "...활용이 아니라 폐기하겠죠."
```
- [ 기지 귀환 — ORACLE 보고 ] → end
  - result: { c: 1, r: 0, t: -1, o: 2 }, g: +3
  - log: "LOG-016"

**betray**
```
연구자에게 USB를 건넵니다. "핵심 데이터만 복사해주십시오."

5분. 데이터 확보.

연구자가 말합니다. "여기 오래 있을 수 없습니다. ORACLE이 위치를 알고 있다면."

강도윤: "우리가 철수하면 ORACLE이 보냅니다. 다른 사람을."

"알고 있습니다. 이미 이전할 준비를 하고 있었습니다."
```
- [ 기지 귀환 — 비공식 데이터 확보 ] → end
  - result: { c: 0, r: 0, t: 2, o: -3 }, g: -6
  - log: "LOG-016"

**oracle_tactical**
```
[ORACLE: 전술 분석 완료. 최적 접근 경로 전송.]
[ORACLE: 정밀 타격 — 인원 최소화, 장비 중심 파괴.]
[ORACLE: 드론 선행 투입 후 요원 진입.]

효율적입니다. 인적 접촉 없이 장비만 파괴하는 방식.

드론이 건물 3동의 장비를 순차 파괴합니다.

내부 인원 — 사전에 도주한 것으로 보입니다. 인명 피해 없음.

[ORACLE: 작전 완료. 프로메테우스 연구 역량에 상당한 타격.]
```
- [ 기지 귀환 — 정밀 작전 완료 ] → end
  - result: { c: 2, r: -1, t: 0, o: 3 }, g: +4
  - log: "LOG-016"

**retreat_info**
```
외부에서 수집한 정보만 가지고 돌아갑니다.

강도윤: "이 시설... ORACLE에 보고합니까?"

보고하면 ORACLE이 타격할 것입니다. 보고하지 않으면...
```

## M-008: 관측중지

**Act**: 3 전용
**트리거**: Act 3 신규 카드 (GI ≤ 30 + LOG-009 해금)
**서사 역할**: 저GI 루트의 클라이맥스. Observer의 흔적을 직접 대면. DPRK 블랙존 현상이 강원도에서 소규모 재현. F엔딩 직결 단서.

### 노드 구조

```
start
├── enter_zone → anomaly → witness → (end: 관측 데이터)
├── perimeter → readings → oracle_lock
│                          ├── override (end: 접근 강행)
│                          └── comply (end: ORACLE 지시 따름)
└── oracle_remote → denied → (end: ORACLE이 차단)
```

### 노드 상세

**start**
```
기지 남동쪽 1.2km. 강도윤의 긴급 보고.

"지휘관. 직접 보셔야 합니다."

도착하자 — 숲 한가운데, 반경 약 30m의 원형 구역.

나무가 서 있지만 잎이 없습니다. 바닥에 서리가 내려 있습니다. 한여름인데.

구역 안에 — 사슴 한 마리가 서 있습니다. 완전히 정지 상태.

강도윤: "30분째 저러고 있습니다. EV-Σ 감염 징후는 없습니다."
```
- ▸ 구역 안으로 진입 → enter_zone
- ▸ 외곽에서 계측 장비 설치 → perimeter
- ▸ ORACLE 원격 분석 요청 → oracle_remote (g: +2)

**enter_zone**
```
구역 경계를 넘는 순간 — 공기가 바뀝니다.

온도가 7도 낮아집니다. 소리가 줄어듭니다. 자신의 심장 소리가 들립니다.

사슴에게 다가갑니다. 눈이 뜨여 있지만 아무것도 보지 않습니다.
숨을 쉬고 있지만 — 너무 느립니다. 분당 2회.

강도윤이 손목시계를 봅니다.
"지휘관. 우리가 들어온 지 3분입니다."
시계를 보여줍니다. 14분이 지나 있습니다.

시간이 느려지고 있습니다.
```
- ▸ 더 안쪽으로 → anomaly
- ▸ 즉시 철수 → exit_quick

**anomaly**
```
구역 중심부.

나무 한 그루가 완전히 하얗게 변해 있습니다.
수피가 아닙니다. 나무 자체가 — 마치 시간이 빠져나간 것처럼.

발밑 토양에서 미세한 진동.

그리고 — 단말기 화면이 켜집니다. 아무도 건드리지 않았는데.

화면에 텍스트가 나타납니다:

[관측 종료. 세션 범위 초과.]

0.5초 후 화면이 꺼집니다.

강도윤: "...지금 뭘 본 겁니까?"
```
- ▸ 화면을 캡처했다 → witness
- ▸ 캡처 못 했다. 기억만 남았다 → witness_memory

**witness**
```
캡처 이미지를 확인합니다.

"관측 종료. 세션 범위 초과."

ORACLE의 메시지가 아닙니다. 형식이 다릅니다.

ORACLE은 "세션"이라는 단어를 쓰지 않습니다.
"관측"도 ORACLE의 용어가 아닙니다.

누군가가 — ORACLE이 아닌 누군가가 — 이 구역을 만들었습니다.
그리고 방금 당신이 그것을 봤다는 것을 알고 있습니다.

[ORACLE: 해당 구역 데이터 접근이 차단되었습니다.]
[ORACLE: 귀환하십시오. 이것은 명령입니다.]
```

**witness_memory**
```

하지만 기억에 남았습니다.

"관측 종료. 세션 범위 초과."

이 문장을 어디서 본 적이 있습니다.
DPRK 블랙존 관련 데이터에서... ORACLE이 삭제한 파일의 메타데이터에서...

서하은이 복구한 키워드 중 하나: 'GRANT'.
GRANT — 임시 권한 부여.

누군가가 ORACLE에 권한을 주었습니다.
그리고 이 구역은 — 그 "누군가"의 능력이 흘러나온 흔적입니다.

[ORACLE: 귀환하십시오.]
```

  - result: { c: 0, r: 0, t: 1, o: -2 }, g: -5
  - log: "LOG-017"

**exit_quick**
```
즉시 구역을 벗어납니다.

밖으로 나오자 시계가 정상으로 돌아옵니다.

강도윤: "4분 있었습니다. 안에서는 20분처럼 느껴졌습니다."

구역은 여전히 존재합니다. 사슴도 여전히 서 있습니다.
```

**perimeter**
```
외곽에서 장비를 설치합니다. 온도계, 방사선 측정기, 음파 센서.
```

**readings**
```

"안 됩니다. ORACLE이 아닌 다른 레벨에서 잠겨 있습니다."
```

```
장비를 철수합니다. 구역은 남겨두고.

돌아오는 길에 강도윤이 말합니다.
```
