# TIU — 정론(Canon) 스토리라인
> 다른 채팅에서 참조용으로 사용되는 마스터 스토리 문서
> 최종 업데이트: 2026-04-12

---

## 1. 세계관 배경

### EV-Σ 사건
- **시대**: 2030년대 초반
- **EV-Σ**: 생물학적 진화 가속 매개체. 프리온 단백질 기반 자기 변형 메커니즘.
- **상황**: 전 세계적으로 확산. 대부분 국가 봉쇄 실패. 필라델피아 하룻밤 만에 붕괴.
- **대한민국**: 봉쇄 성공률 97.3% — 전 세계 유일 95%+ 구역.
- **ORACLE 평가**: *"This region should have failed. Outcome does not match model."*
- **비밀**: 성공률의 31%는 프로메테우스 비공식 기술 지원 (COASTAL MIRROR 작전).

### ORACLE 시스템
- **표면**: "자율 관리 네트워크". 전 세계 격리 기지 원격 통제.
- **실체**: 한국의 기밀을 수집하며 동시에 봉쇄를 유지하는 이중 구조.
- **기능**: 지휘관 성과 평가, 보급 관리, 작전 권고, 데이터 수집.
- **UI**: 게임 전체가 ORACLE 단말기 인터페이스. 모든 텍스트가 ORACLE의 목소리.
- **핵심 비밀**: 매일 02:00~04:00 본부로 한국 군사 네트워크, 해안 방벽 사양, 요원 충성도 데이터 전송.

### Observer
- ORACLE 아래, 너머, 이전에 존재하는 무언가.
- 게임의 "회차"는 Observer가 ORACLE에 부여한 임시 능력(GRANT)으로 가능.
- Observer는 유도 지수(GI)를 통해 선택지 방향성을 조율.
- 플레이어가 이 구조를 인지하는 순간이 게임의 진짜 클라이맥스.

### 한국 지부 (KR-INIT-001)
- **코드**: BRANCH KR-INIT-001
- **위치**: 강원도 내 비공개 위치
- **규모**: 소규모, 인원 10명 내외
- **설립**: Observer와 ORACLE이 한국 방어 성공에 주목하여 설립
- **특이 이력**: 초대 지휘관 부임 전 3개월간 ORACLE 단독 운영

## 2. 주요 인물 프로필

### 이중철 (Pilehead) — 플레이어 캐릭터
- **전직**: 육군 특수부대 장교 (대위 추정). 불명예전역.
- **전역 원인**: 프로메테우스 관계자(닉 포스터)가 부대원 조롱 → 과격 대응 → 불명예전역.
- **현재**: ORACLE Proxy Network 한국 지부 초대 지휘관.
- **설계 의도**: 개인 감정과 ORACLE의 "프로메테우스 = 위협" 분류가 우연히 일치 → 자발적 복종.

### 서하은 — 부지휘관 / 데이터분석
- **직책**: 부지휘관, ORACLE 데이터 해석 및 브리핑.
- **배경**: 지휘관 부임 전 3개월간 ORACLE 지시만으로 기지 운영.
- **Act 1**: 자기소개 — "오라클 한국지부의 부지휘관 및 데이터 분석을 맡고 있는 서하은".
- **Act 2**: ORACLE 데이터 불일치 최초 감지. "같은 상황인데 다른 권고가 내려올 때가 있었습니다."
- **Act 3**: ORACLE 데이터 삭제 흔적 발견, 복구 작업 시작.
- **분기**: ORACLE이 전출 명령 자동 발령 → 잔류/전출이 Act 4 전체를 결정.
- **잔류 시**: 삭제 데이터 복구 완료 → F/D/B 엔딩 루트 접근 용이.
- **전출 시**: 정보 분석 역량 급감, 박소영이 후임.

### 강도윤 — 전술지휘관 / 현장작전
- **전직**: 해병대 부사관. 과묵하고 실용적.
- **역할**: 봉쇄선 순찰, 이변체 대응 작전 지휘, 이중철의 현장 파트너.
- **Act 1**: 자기소개 — "전술지휘관 강도윤입니다. 현장작전 및 봉쇄선 관리를 담당하고 있습니다."
- **특기**: 서하은 전출 저지 시 합류. 봉쇄선 외곽 미확인 흔적 추적.

### 윤세진 — 연구원 / 의료관
- **배경**: 대학원 프리온 단백질 연구자. EV-Σ는 한국지부에서 첫 접촉.
- **역할**: 이변체 표본 분석, 부상 치료, EV-Σ 억제제 개발.
- **핵심 설정**: 오라클 데이터를 보완하는 현장 관측 기반 분석 기법 개발.
- **Act 1**: 자기소개 — "연구원 겸 의료관 윤세진입니다. EV-Σ 생물학 연구와 기지 의료를 담당하고 있어요."
- **업적**: Phase 0→1 전환을 40% 지연시키는 억제제 화합물 발견.

### 임재혁 — 정보분석관 / 기술관
- **역할**: 기지 보안 시스템, 정보 수집·분석, ORACLE 연동 및 통신 체계 전반 관할.
- **성향**: 팀 내 ORACLE 의존도 최고. 신뢰도 상승 시 ORACLE 이상 현상 공유.
- **Act 1**: 자기소개 — "정보분석관 임재혁입니다. 기지 보안 시스템과 정보 수집·분석을 담당합니다."
- **발견**: ORACLE 아키텍처 내 비공개 레이어(5계층 이상), 외부 데이터 전송 추적.

### 프로메테우스 인물
- **마르쿠스 베버**: 작전팀 지휘관. 독일어 억양. 차분하고 논리적. 봉쇄 성공률 31% 기여 증거 보유.
- **닉 포스터**: 이중철 불명예전역 사건 당사자. 기지 주변 활동. 전술 장비 착용.
- **박소영**: 민간 데이터 분석관. 서하은의 추천. 서하은 전출 시 후임 가능.
- **에이전트 강 (강지현)**: 정체 미공개. ORACLE도 프로메테우스도 아닌 제3의 관찰자. Act 2부터 흔적만.

## 3. 4-Act 스토리 흐름

### Act 1: 프롤로그/일상 (~5일)
**톤**: 차분, 의례적
**핵심 이벤트**:
- CA-001 "기지 도착 첫날" — 게임 시작 시 반드시 첫 카드로 강제 배치.
- ORACLE 단말기 최초 활성화. GRANT 부여 확인.
- 간부진 자기소개 이브닝 챗 — **ORACLE 의심 없음**, 순수한 인사+업무보고 톤.
- 이변체 조우 없음. 현장임무(FIELD MISSION) 미발생.
- once 카드: 소개카드 중복 방지.
**전환**: ~5일 경과 → Act 2 (루트 A 고정)

### Act 2: 의혹 (~14일)
**톤**: 안정→불안, ORACLE 의심 시작
**핵심 이벤트**:
- 이변체 첫 조우 (Shell Talker, Blood Pit).
- 프로메테우스 첫 감지.
- **ORACLE 의심 시작**: 서하은 "ORACLE 지시가 일정하지 않다", 윤세진 "예측과 관측 오차 방향이 항상 같다", 임재혁 "시스템 구조가 이상하다".
- 연쇄: CH-001 (서하은의 조사), CH-002 (봉쇄 위기), CH-003 (ORACLE의 시험).
**전환**: ~14일 경과 → Act 3 (4루트 분기: A/B/C/D)
- A: 프로메테우스 접촉 + 미션 완료 (페널티 0)
- D: 둘 다 안 함 (페널티 -4)

### Act 3: 진실 (~29일)
**톤**: 긴장, 피로, ORACLE 실체 접근
**핵심 이벤트**:
- 프로메테우스 직접 대면. 닉 포스터 등장.
- 서하은의 ORACLE 데이터 삭제 흔적 발견 (C-071).
- 임재혁의 ORACLE 외부 접속 시도 발견 (C-079).
- ORACLE의 지휘관 모니터링 강화 (C-072, C-076, C-078).
- 에이전트 강 흔적 등장 시작.
- 연쇄: CH-005 (프로메테우스 접선), CH-006 (서하은의 마지막 밤).
- 일일 감쇠: c-1, r-1
**전환**: ~29일 경과 → Act 4 (4루트 분기 + 전체 -5 페널티)

### Act 4: 최종 (Day 30+)
**톤**: 위기, 선택, 불가역
**핵심 이벤트**:
- **C-073 서하은 전출 명령** (ORACLE 자동 발령) — 게임의 핵심 분기점.
- 잔류 선택: CS-001→003 저지 루트 → 삭제 데이터 복구 → 엔딩 접근 용이.
- 전출 수용: 정보 분석 역량 급감, 박소영 후임.
- ORACLE 최종 프로토콜 발동 조짐.
- 마르쿠스 베버의 핵심 증거 공개.
- 일일 감쇠: c-2, r-2, t-1

## 4. ORACLE의 진실

### 표면 → 실체
1. **표면**: 효율적인 자율 관리 시스템. 지휘관을 돕는 도구.
2. **Act 2 발견**: 데이터 스트림의 선별적 지연. 같은 상황에서 다른 권고.
3. **Act 3 발견**: 로그 삭제 흔적 (키워드: PROMETHEUS, COASTAL, GRANT). 요원 충성도 모니터링.
4. **Act 4 발견**: 매일 02:00~04:00 한국 군사 데이터를 본부로 전송. 한국을 보호하는 것이 아니라 감시.

### ORACLE이 삭제한 데이터 (LOG-018, 서하은 최종 보고서)
1. 프로메테우스 COASTAL MIRROR 작전 정보
2. 한국 해안 방벽 기술 데이터 전송 기록
3. GRANT 임시 권한 전체 내용
4. 지부 설립 이전 한국 관련 ORACLE 활동 기록

### Observer의 정체
- ORACLE 너머에 존재. ORACLE도 인식하지 못하는 레이어.
- 회차 플레이 = Observer가 ORACLE에 부여한 GRANT 능력.
- GRANT 만료 조건: `UPON_FULL_ESTABLISHMENT` || `OBSERVATION_TERMINATE`
- F엔딩에서 직접 접촉: *"OBSERVATION SUSTAINED."*

## 5. 엔딩 분기 (8종)

### 즉시 엔딩 (스탯 0)
C_c(봉쇄0), C_r(자원0), C_t(신뢰0), C_o(평가0) — 즉시 게임오버.

### 특수 엔딩
| 엔딩 | 조건 | 내러티브 |
|------|------|----------|
| **A. 완벽한 도구** | 봉쇄100 + GI>=60 | ORACLE의 이상적 운용자. 완벽한 복종. |
| **B. 각성** | Act4, GI<=-15, 신뢰65+ 2명, LOG6+ | 진실의 일부를 보았다. 이전처럼 따를 수 없다. |
| **D. 조용한 자유** | Act4, GI<=-30, 신뢰60+ 3명, LOG8+ | 새벽 4시, 팀 전체가 기지를 떠났다. |
| **F. 데이터 손상** | Act4, LOG-012+013, GI<=0 | Observer 직접 접촉. 단말기가 꺼진다. |
