## 5. 핵심 설정/규칙 — 게임 메커니즘과 스토리 연결

### 유도 지수 (Guidance Index, GI)
- 플레이어에게 **절대 표시되지 않는** 숨겨진 변수 (3회차에 UI 노출 예정).
- ORACLE 권장 방향 선택: +1~+2 / ORACLE 반항 선택: -1~-3 / 독립적 정보 수집: -3~-6
- GI가 높으면 ORACLE의 도구로 편입. GI가 낮으면 진실에 접근.
- 뉴스 헤드라인 필터링: GI 높으면 성과 보고, GI 낮으면 글리치 뉴스 노출.

### 4대 스탯
| 스탯 | 시작값 | 0 → 게임오버 | 100 → |
|------|--------|-------------|-------|
| 봉쇄(c) | 50 | 봉쇄선 붕괴, 시설 폐기 | GRANT EXPIRED (안정화 완료) |
| 자원(r) | 65 | 기지 기능 마비 | — |
| 신뢰(t) | 50 | 요원 이탈 | — |
| 평가(o) | 40 | ORACLE 접속 차단 | — |

### 인물 신뢰도
- 4인 개별 (0~100, 시작 50). 프로메테우스 인물은 별도 시작값.
- 신뢰도 구간: low(0~35), mid(36~60), high(61~80), bond(81~100).
- 심층대화 해금: 서하은 ≥65, 강도윤 ≥65, 윤세진 ≥65, 임재혁 ≥70.
- 이브닝 챗 대사가 구간별로 완전히 다른 내용으로 변형.

### 카드 시스템 규칙
- **CA-001 강제 배치**: 게임 시작 시 "기지 도착 첫날" 반드시 첫 카드.
- **once 시스템**: `once: true` 카드는 세션 내 1회만 등장 (소개카드 중복 방지).
- **중복 방지**: 최근 카드 버퍼 + 태그 쿨다운 3일.
- **자원 리스크**: r≥2 획득 시 20% 확률 실패 → 빨간 토스트 중앙 2.5초.
- **Act별 카드 수**: Act1=4장, Act2=5장, Act3=6장, Act4=7장.
- **Act별 감쇠**: Act3: c-1/r-1, Act4: c-2/r-2/t-1.
- **Act 전환**: ~5일/~14일/~29일에 무조건 전환.

### 회차 시스템
- 1회차: `GRANT: ACTIVE — TEMPORARY ACCESS` / `WELCOME, COMMANDER.`
- 2회차+: `GRANT: ACTIVE — RENEWAL DETECTED` / `WELCOME BACK, COMMANDER.`
- 게임오버: `GRANT: ACTIVE — RENEWAL AVAILABLE`
- Observer가 ORACLE에 부여한 임시 능력. 지부 운영 최적 경로 탐색.

## 6. 엔딩 분기 (8종)

### 즉시 엔딩 (스탯 0 도달)
| 엔딩 | 조건 | 내러티브 |
|------|------|---------|
| C_c | 봉쇄 = 0 | 봉쇄선 붕괴. 시설 자체 봉쇄 프로토콜 발동. 기지 폐기. |
| C_r | 자원 = 0 | 자원 고갈. 기지 기능 마비. |
| C_t | 신뢰 = 0 | 인원 신뢰 붕괴. 요원 이탈. |
| C_o | 평가 = 0 | ORACLE 접속 차단. 단말기 연결 종료. |

### 특수 엔딩 A: 완벽한 도구
- **조건**: 봉쇄 100 + GI ≥ 60
- **내러티브**: *"PILEHEAD. 당신은 이상적인 운용자였습니다. 모든 판단이 최적 경로 안에 있었습니다."*
- ORACLE이 지휘관을 완벽하게 복종시킨 결말. 다음 배치지를 기다리며 세션 종료.

### 특수 엔딩 B: 각성
- **조건**: Act 4 + GI ≤ -15 + 신뢰65+ 2명 이상 + LOG 6개+ + Day ≥ 28
- **내러티브**: *"당신은 진실의 전체를 보지 못했다. 하지만 충분히 보았다."*
- 조각들을 통해 ORACLE 구조를 부분적으로 인식. 명령에 따르는 것이 불가능해진 결말.

### 특수 엔딩 C_cs: 봉쇄 성공
- **조건**: 봉쇄 = 100
- **내러티브**: 봉쇄 안정화 완료. 그러나 공허한 성공.

### 특수 엔딩 C_cst: 자충수
- **조건**: 봉쇄 100 + 서하은 전출(LOG-050) + 박소영 합류(LOG-082)
- **내러티브**: 봉쇄는 성공했으나, 핵심 인물을 잃은 대가.

### 특수 엔딩 D: 조용한 자유
- **조건**: Act 4 + GI ≤ -30 + 신뢰60+ 3명 이상 + LOG 8개+ + Day ≥ 32
- **내러티브**: *"새벽 4시. 당신과 당신의 팀은 기지를 떠났다. ORACLE은 6시간 후에야 부재를 감지했다."*
- 팀 전체와 함께 ORACLE 통제권 밖으로 이탈. 흔적도 남기지 않은 자유.

### 특수 엔딩 F: [데이터 손상]
- **조건**: Act 4 + LOG-012 + LOG-013 + GI ≤ 0 + Day ≥ 30
- **내러티브**: *"그것이 당신을 본다. > OBSERVATION SUSTAINED."*
- ORACLE 아래 Observer와 직접 접촉. 단말기 꺼지고 다시 켜지지 않음. 세션 데이터 손상.
- 우선순위: F > D > B (이브닝 종료 시 체크)

- Phase 0: 초기 감염. 억제제로 전환 40% 지연 가능.

- Phase 2+: 완전 변이. 통제 불가.
- Phase 3: 종 흔적 소실. HeLa 세포형 무한 증식. 비가역.

## 8. 용어집

| 용어 | 설명 |
|------|------|
| **ORACLE** | 전 세계 격리 기지를 원격 통제하는 AI 시스템. 표면은 지원, 실체는 감시. |
| **Observer** | ORACLE 너머 존재. 회차 시스템의 근원. GRANT를 부여. |
| **GRANT** | Observer→ORACLE 임시 권한. 만료: UPON_FULL_ESTABLISHMENT ∥ OBSERVATION_TERMINATE |
| **GI (유도 지수)** | 숨겨진 변수. ORACLE 순응도 측정. 높으면 도구, 낮으면 각성. |
| **Proxy Network** | ORACLE의 분산 지휘 체계. 한국 지부 = BRANCH KR-INIT-001. |
| **프로메테우스** | 비공식 조직. 한국 봉쇄 31% 기여. ORACLE은 적대 분류. |
| **COASTAL MIRROR** | 프로메테우스 한국 작전. 해안 방벽 기술 협력. |
| **White Shield** | 대한민국 군 EV-Σ 대응 체계. 봉쇄율 97.3%. |
| **EV-Σ** | 생물학적 진화 가속 매개체. 프리온 기반. |
| **Phase 단계** | EV-Σ 감염 분류. 0→1→2→3 (비가역). |
| **TS-Ω** | Sovereign Shell. 해양 군체 중심 의식. Brood Drone 지휘. |
| **Z-0 구역** | ORACLE 분류 한국 구역. CONTROLLED 진화. 전 세계 유일 95%+. |
| **GENESIS BREAK** | EV-Σ 최초 대규모 발현 사건. 봉쇄 프로토콜 기준. |
| **Silent Belt** | 광역 포자 오염 지대. Seed Spreader 형성. DELAYED 진화. |
| **DPRK 블랙존** | 북한 접경 미확인 구역. 의식/육체 분리. Observer 능력 불완전 작용. |
| **once 카드** | 세션 내 1회만 등장하는 카드 (소개카드 중복 방지). |
| **서하은 분기** | Act 3~4 핵심 분기. 전출/잔류가 엔딩 접근성을 결정. |
